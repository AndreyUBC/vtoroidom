Как пользоваться панелью и публиковать изменения на сайте
📖 Как обновить сайт — пошагово
  1. Откройте нужный раздел в левом меню и отредактируйте текст или фото.
  2. Нажмите кнопку «Сгенерировать код» — откроется окно с готовым HTML.
  3. Нажмите «Скопировать».
  4. Откройте github.com → ваш репозиторий.
  5. Найдите нужный файл (например, index.html), кликните по нему.
  6. Нажмите значок карандаша ✏️ (Edit this file) в правом верхнем углу.
  7. Выделите весь текст (Ctrl+A) и вставьте скопированный код (Ctrl+V).
  8. Нажмите Commit changes — сайт обновится автоматически через 1–2 минуты.
🖼 Как добавить своё фото
  1. В GitHub-репозитории откройте папку img/ (или создайте её).
  2. Нажмите Add file → Upload files и загрузите фотографии.
  3. Скопируйте относительный путь, например: img/room1.jpg
  4. В этой панели откройте «Фото — Вагапова» или «Фото — Кутуя».
  5. Вставьте путь в поле «URL новой фотографии» и нажмите + Добавить.
  6. Нажмите «Сгенерировать код photos.html» и опубликуйте, как описано выше.
🔐 Безопасность

Пароль хранится в браузере (localStorage). Эта панель — только для вас: она не подключена к интернету и не отправляет данные никуда. Если откроете сайт с другого компьютера, нужно будет снова установить пароль через раздел «Пароль».
